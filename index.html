Dados e metodologia: Open-Meteo — modelos ECMWF IFS 0.25°, ICON Seamless, GFS Seamless. Consensus de rajadas = média dos máximos por modelo (excluindo modelos sem dados de rajadas). O campo "Pior" mostra o máximo absoluto entre modelos — útil como margem de segurança para içamentos. Vento e humidade = média simples dos modelos. Precipitação = média dos acumulados. Fontes secundárias: Yr.no (MET Norway) · IPMA (Angra do Heroísmo proxy ilha). Janelas de trabalho: 07h–13h e 14h–17h (hora Açores = UTC−1).
